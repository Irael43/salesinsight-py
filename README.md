# SalesInsight PY - Pipeline de Análise de Dados de Vendas

## 📌 Sobre o Projeto

Pipeline completo de análise de dados de vendas desenvolvido em Python para o Módulo 1 do curso de Desenvolvedor(a) em IA para Análise Preditiva.

## 🚀 Funcionalidades

- Geração/Leitura de dataset de vendas
- Limpeza e tratamento de dados
- Criação de colunas derivadas
- Métricas agregadas com groupby
- Segmentação de clientes (Bronze, Prata, Ouro)
- Estatísticas com NumPy
- Visualizações com Matplotlib e Seaborn
- Pipeline com classe e herança
- Exportação CSV e JSON
